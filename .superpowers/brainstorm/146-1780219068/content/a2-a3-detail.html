📱 A3 · PWA 离线支持 — 详细设计

Next.js + next-pwa 插件，类 App 体验

📴 离线训练能力

Service Worker 缓存静态资源 + 已加载的训练数据。离线状态下仍可完成当日训练队列，联网后自动同步进度。

离线可用

  • 已缓存的训练队列
  • 单词浏览和搜索
  • 错题本查看
  • 训练进度本地暂存

需联网

  • 新笔记导入 / AI 提取
  • TTS 首次加载语音
  • 数据同步到云端
  • 统计面板最新数据

🏠 安装到桌面 / 手机

满足 PWA 安装条件后，浏览器提示"添加到主屏幕"，体验接近原生 App。

  • 🖥️ 桌面端：独立窗口，无浏览器地址栏
  • 📱 移动端：添加到手机主屏幕，全屏运行
  • 🔔 推送通知：每日训练提醒（通过 Notification API）
  • 🎨 自定义图标 / 启动画面 / 主题色

🔄 离线同步策略

联网时正常操作；离线时本地 IndexedDB 暂存，联网后自动合并。

在线模式
API → Supabase
数据实时同步
离线模式
IndexedDB 暂存
训练记录标记为待同步
恢复在线
自动 merge
冲突以服务器为准

⚠️ 注意事项

  • PWA 离线同步增加了复杂度，建议作为 Phase 2 功能，第一版先做纯在线
  • iOS Safari 对 PWA 支持有限（推送通知不支持），但"添加到主屏幕"可用
  • 离线数据量需控制——单次缓存不超过用户最近 7 天的训练内容
