# English Lab · 设计规格文档

> 日期：2026-05-31  
> 项目：自用英语学习网站  
> 状态：设计完成，待进入开发实施

---

## 1. 项目概述

一款对接 NotebookLM 的个人英语学习工具。从笔记中提取词汇，按艾宾浩斯遗忘曲线安排复习，区分阅读/口语/写作三种训练模式。

## 2. 技术栈

| 层面 | 选型 |
|------|------|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS + shadcn/ui |
| 数据库 | Supabase (PostgreSQL) |
| 认证 | Supabase Auth (邮箱登录) |
| AI 提取 | NotebookLM API + Gemini API |
| 部署 | Vercel (免费版) |
| 字体 | Fraunces + DM Sans + DM Mono (Google Fonts) |

## 3. 模块架构

### 核心模块（6 个）

| # | 模块 | 职责 |
|---|------|------|
| M1 | 笔记导入与AI提取 | 四步流程：选笔记 → AI提取 → 审核确认 → 入库 |
| M2 | 词汇与内容管理 | 单词/短语/句式 CRUD，搜索筛选，来源追溯 |
| M3 | 艾宾浩斯调度引擎 | 遗忘曲线 1/2/4/7/15/30 天间隔，每日训练队列 |
| M4 | 训练执行模块 | 阅读（闪卡+翻译+填空+理解）/ 写作 / 口语 |
| M5 | 进度追踪与统计 | 热力图、掌握率、词汇量趋势、日历 |
| M6 | 错题强化闭环 | 自动收录 → 短间隔重排 → 连续正确释放 |

### 辅助模块（3 个）

| # | 模块 | 说明 |
|---|------|------|
| A1 | 目标设定与提醒 | 每日目标 + 浏览器通知 + 连续打卡 |
| A2 | TTS 语音增强 | Web Speech API 发音（Phase 4） |
| A3 | PWA 离线支持 | Service Worker + 离线训练（Phase 4） |

## 4. 设计系统

### 4.1 设计语言

**杂志编辑风格（Editorial / Magazine）** — 安静、精致、书卷气。字体层级驱动，暖白纸色底色，极弱投影，大量留白。

### 4.2 色彩体系

| Token | 值 | 用途 |
|-------|-----|------|
| `--paper` | `#faf7f2` | 页面底色 |
| `--card` | `#ffffff` | 卡片容器 |
| `--ink` | `#1a1a1a` | 主文字 / 深色块 |
| `--body` | `#4a4a4a` | 正文 |
| `--muted` | `#8a8a8a` | 次要文字 |
| `--faint` | `#c4c0ba` | 辅助/禁用文字 |
| `--rule` | `#e8e4de` | 分割线/边框 |
| `--charcoal` | `#1e1e1e` | 深色容器背景 |
| `--gold` | `#c4a030` | 数据强调（深色底上） |
| `--accent` | `#3b6e8e` | 数据强调（浅色底上） |

### 4.3 字体层级

| 用途 | 字体 | 字重 | 字号 |
|------|------|------|------|
| 封面头条 | Fraunces italic | 800 | 88px |
| 页面标题 | Fraunces italic | 700 | 26–30px |
| 数据大字 | Fraunces italic | 700–800 | 36–72px |
| 卡片标题 | DM Sans | 600 | 14px |
| 正文 | DM Sans | 400–500 | 13–15px |
| 辅助/标签 | DM Sans | 500–600 | 10–12px |
| 音标/代码 | DM Mono | 400–500 | 11–15px |

### 4.4 形态语言

- **卡片**：圆角 14px，1px 暖灰边框，多层极弱投影
- **按钮**：圆角 8–12px，主按钮炭黑填充，次按钮细线边框
- **分割线**：1px `--rule`，宽度 40px 或全宽
- **留白**：页面上下各 6vh，段间距 20–48px

### 4.5 投影系统

| 层级 | CSS |
|------|-----|
| 基础卡片 | `0 1px 2px rgba(0,0,0,0.03), 0 2px 8px rgba(0,0,0,0.02)` |
| 提升卡片 | `0 1px 3px rgba(0,0,0,0.04), 0 4px 16px rgba(0,0,0,0.03)` |

## 5. 页面设计

### 5.1 首页（Hero 封面式）

- 左对齐杂志封面布局
- 刊名 Masthead → 头条 Headline（88px Fraunces）→ 副线 Cover lines → CTA → 目录栏 Folio
- 双栏不对称：左 1fr 主故事，右 1fr 三条数据副线

### 5.2 笔记导入页

- 四步流程：Select → Extract → Review → Import
- 顶部步骤指示器，斜体数字 + 横线
- 列表式信息展示，每条可勾选/取消

### 5.3 统计页

- 三栏布局：左导航 200px + 中核心数据 + 右辅助 260px
- 顶部深色 Hero Banner 作为视觉锚点
- 中部：年度训练热力图 + 掌握率 + 复习队列
- 右侧：日历 + 今日进度环 + 快捷操作

### 5.4 训练页（待统一）

- 当前旧版漫画风格，需统一为杂志风
- 保留闪卡核心交互：看到单词 → 翻面 → 自评

## 6. 开发分阶段

| Phase | 内容 | 预计 |
|-------|------|------|
| 1 | 项目搭建 + 数据模型 + 内容管理 + 笔记导入 | 1–2 周 |
| 2 | 艾宾浩斯引擎 + 阅读训练 + 错题闭环 | 2–3 周 |
| 3 | 写作训练 + 口语训练 + 统计面板 + 目标提醒 | 2–3 周 |
| 4 | TTS 语音 + PWA + 最终打磨 | 2–3 周 |
